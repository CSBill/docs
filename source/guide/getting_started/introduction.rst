============
Introduction
============

CSBill is an open-source application that allows you to manage clients and contacts and send invoices and quotes.

-------------------
System Requirements
-------------------

.. note::
    In order to run CSBill, you need at least PHP version **5.6.10**, but the latest version is always recommended

**Requirements:**

* PHP (>=5.6.10)
* MySQL Database (> 5.5.0)
